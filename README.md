# Multichain Wallet Asset Tracker

This skill detects and displays crypto assets on multiple blockchains including:
- Ethereum (ETH, USDT)
- BSC (BNB, CAKE)
- Polygon (MATIC, USDC)
- Solana

## Features
- Supports multiple chains
- Simple and lightweight
- Easily extendable
